# Romaji-English-IPA
（整活）英语罗马字音标，更适合中国人的英语学习工具

## 简介
还在为看不懂英语国际音标而苦恼吗？别担心，这个项目就是为你准备的！我们将英语单词按照罗马字的方式进行分类，并且配上对应的国际音标，让你轻松掌握英语发音，提升你的英语水平！

## 使用前提
1. 已学习汉语拼音
2. 你会日语五十音（包括清浊音、拗音）
